## üõ†Ô∏è Getting Started

### Prerequisites

*   **Node.js 18+**
*   **MongoDB**

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/DionathaGoulart/link-shortener--Angular.git

# 2. Setup Backend
cd backend
npm install
npm start

# 3. Setup Frontend (in a new terminal)
cd ../frontend
npm install
npm start
```
